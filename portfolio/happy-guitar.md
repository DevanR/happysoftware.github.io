# Happy Guitar

**Interactive guitar fretboard and chord trainer with audio feedback and XP progression.**

[Visit Happy Guitar](https://guitar.happy.software)

## Overview

Happy Guitar is a web-based learning tool designed to help guitarists master the fretboard, chords, and music theory through interactive exercises and gamification.

## Key Features

- **Fretboard Notes** - Identify notes anywhere on the guitar fretboard with timed challenges
- **Common & Barre Chords** - Learn and practice open and barre chord shapes
- **Triad Master** - Practice triads across different inversions and string sets
- **Audio Feedback** - Hear notes and chords via Web Audio API for ear training
- **XP System** - Track your progress with levels and experience points

## Technologies

- Vite
- TypeScript
- Web Audio API
- CSS3
